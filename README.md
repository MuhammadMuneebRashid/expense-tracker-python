# Expense Tracker

A simple command-line Expense Tracker built with Python. This project uses file handling to store and manage expense records in a text file. It is designed to demonstrate the use of Python functions, file operations, and a menu-driven interface.

## Features

* Add new expense records
* View all saved expenses
* Search expenses by category
* Delete expenses by category
* Calculate the total expense amount
* Store data persistently using a text file

## Technologies Used

* Python 3
* File Handling
* Functions
* Conditional Statements
* Loops

## Project Structure

```text

```

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the Python file:

```bash
python expense_tracker.py
```

4. Use the menu to manage your expenses.

## Sample Menu

```text
=== Expense Tracker ===
1. Add Expense
2. View Expense
3. Search Expense
4. Delete Expense
5. Total Expense
6. Exit
```
